Golang Label

  1. Label 使用

    1. Label 只能与 goto break continue 共同使用

    2. 使用限制：

      1. 关键字 使用范围 效果
        goto 没有任何使用限制 跳转到任何地方
        break  label只能在循环(for switch)紧挨的上方 终止循环，不再进入循环
        continue label只能在循环(for switch)紧挨的上方 暂停本轮循环

    3. 例如：

      func main() {
a := 1
LABEL1:

println("LABEL1")
println("inc a=", a)
a += 1

LABEL2:
 println("LABEL1") // 会报错的

for a < 6 {
println(a)
if a == 3 {
a += 1
continue LABEL2
}
if a == 5 {
break
}
goto LABEL1
}

}

    4. 经典用法:
      平常的循环嵌套：

      for _, item := range list.Items {
found := false

for _, reserved := range reserved.Items {
if reserved.ID == item.ID {
found = true
break
}
... do some other work ...
}

if found {
continue
}

... do some other work ...
}

      使用label的嵌套:

      OUTER:
for _, item := range list.Items {
for _, reserved := range reserved.Items {
if reserved.ID == item.ID {
    continue OUTER
}
... do some other work ...
}
... do some other work ...
}

      Label放在for循环之前是表明跳出到哪一层的循环，不会影响到for循环，并不是每次跳跃之后都要重新执行for循环
